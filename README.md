## 特性

- 🎉 利用 Miru 的扩展系统，可以轻松的添加新的源
- 🦋 优秀的 UI 设计与交互体验。
- 📱  支持移动端浏览器。
- 🌐 支持自定义扩展仓库与代理。
- 📦 官方扩展仓库提供视频源，无需编写任何扩展即可使用。
- 📺🖼️📄 支持在线观看多种视频、漫画、小说源，实现多平台的统一。
